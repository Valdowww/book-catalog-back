# Catalogue de Livres - Application CRUD Full-Stack

## Présentation du Projet
Ce projet est une application de gestion de catalogue de livres réalisée dans le cadre du cours "Création d'application".

## Instructions de Lancement

### 1. Back-end (Spring Boot)
Le serveur expose une API REST sur le port `8080`.
1. Naviguer dans le dossier du projet bookcatalog.
2. Lancer l'application avec Maven :
   ```bash
   mvn spring-boot:run

### 2. Front-end (Angular)
L'interface utilisateur tourne dans le navigateur sur le port `4200`.
1. Naviguer dans le dossier du projet book-catalog-front.
2. Installer les dépendances :
    ```bash
   npm install
3. Lancer le serveur :
    ```bash
   ng serve
4. Ouvrir un navigateur à l'adresse : http://localhost:4200
